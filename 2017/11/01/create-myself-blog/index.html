create myself blog

对于开发人员拥有一个自己的博客，可以积累，方便以后找工作或跳槽

开始搭建

第一步 安装全局hexo

hexo.io 官网
安装 hexo

第二步：初始化hexo项目

hexo init blog #blog 是文件夹名字，随便起
cd blog
npm install

hexo d

第三步：创建一个仓库

第四步:把项目以git-page分支提交到远程仓库

项目git初始化
cd blog/public
在这个文件夹地下初始化

git init
git add .
git commit -am"init"
git push origin master

git checkout -b  git-pages
git push origin git-pages

在github 上面如果有git-pages分支，这个仓库就会开启，git page

然后就可以使用 
https://wsdo.github.io/ 这个域名来访问

hexo 使用

hexo new "文章的标题"
在项目里面打开 source/_posts/ 里面的md文件就是要写文章

hexo g #把自己写文章生成html静态文件

然后推送到github 让大家访问

可以配置hexo deploy 直接发布到git

可以自己配置主题 插件
